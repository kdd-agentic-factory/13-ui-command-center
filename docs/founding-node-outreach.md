# KDD Founding Nodes — Outreach Playbook

## Goal
Land 20–30 high-quality founding nodes for KDD Knowledge Network.

## Positioning

We are not sending a generic product launch.
We are inviting a small group to join the first private nodes of KDD.

Core line:

> Tus datos siguen siendo tuyos, pero tu aprendizaje se acelera porque formas parte de una red que aprende.

## Priority order

### 1. Track-day riders

Best first fit.

Why:

- clear pain
- sessions repeat
- video / times / telemetry already exist
- they want a simple next-step plan

### 2. Ride organizers / coaches

Best commercial channel.

Why:

- already have a community
- can offer KDD as a differentiator
- can validate with multiple riders at once

### 3. Schools / circuits

Best controlled pilot.

Offer:

- 1 day
- 10 riders
- 1 circuit
- 1 report per rider
- 1 aggregate coach report

### 4. Teams / academies

Higher-value professional nodes.

Message:

- multi-rider
- setup traceability
- private/team/federated modes

### 5. MotoStudent / universities

Strong technical fit.

Why:

- engineering + data + simulation
- useful for documentation and validation

## Outreach sequence

### Stage 1 — Warm contacts

Goal: 5–10 quick calls / demos.

### Stage 2 — Organizers and coaches

Goal: 2 pilot agreements with real riders.

### Stage 3 — Academies / teams / universities

Goal: higher-quality nodes and stronger case studies.

## Messaging rules

- keep it short
- do not send a giant PDF first
- ask for a 15-minute demo
- focus on one outcome: better next-tanda decisions

## Suggested subject lines

- Invitación privada · KDD Founding Node
- Early access para pilotos de circuito
- Red privada de conocimiento para rendimiento en moto
- Queremos probar KDD con pilotos reales

## Short WhatsApp / DM script

Hola, estoy abriendo el early access de KDD Moto Intelligence, una plataforma de telemetría e IA para pilotos de moto en circuito.

La idea es simple: subes una sesión y KDD detecta dónde pierdes tiempo, qué patrón se repite y qué deberías trabajar en la siguiente salida.

Además, estamos creando una red privada de conocimiento: tus datos se protegen, pero si decides conectarte como nodo, la red aprende de patrones agregados y tú recibes mejores recomendaciones.

Estoy buscando los primeros pilotos Founding Node para probarlo con sesiones reales.

¿Te enseño una demo de 15 minutos?

Hola,

Además, estamos construyendo una red privada de conocimiento para que los nodos aprendan más rápido sin exponer datos crudos.

Busco pilotos Founding Node para probarlo con sesiones reales.

¿Te enseño una demo de 15 minutos?

## Email template — coaches / organizers

Asunto: KDD para vuestra jornada / curso

Hola,

Estoy abriendo el early access de KDD Moto Intelligence para rodadas, cursos y jornadas de pilotaje.

La idea es convertir cada tanda en un informe útil:

- dónde se pierde tiempo
- qué curva concentra el margen de mejora
- qué trabajar en la siguiente salida
- cómo evolucionan los pilotos entre sesiones

Podemos probarlo con un grupo pequeño y generar informes individuales + un informe agregado para el coach o la escuela.

¿Agendamos una demo breve?
