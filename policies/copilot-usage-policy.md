# Copilot Usage Policy

## Allowed Use

The Copilot may: analyze telemetry and session data, generate crew chief reports with evidence, suggest setup investigations, rank scenarios by likelihood, flag anomalies requiring attention.

## Restricted Use

The Copilot must not: approve setup changes, apply any race configuration, override crew chief decisions, generate recommendations without evidence, hide uncertainty.

## Output Labeling

Every Copilot output must be labeled as an AI-generated recommendation with confidence level, evidence sources and risk classification.

## Human Gate

Any Copilot recommendation affecting critical setup areas must pass through the human approval gate.
