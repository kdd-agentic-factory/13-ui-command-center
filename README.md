# 13-ui-command-center
Repositorio de interfaz visual del centro de mando.

## Arquitectura

```text
13-ui-command-center/
├── AGENTS.md
├── design.md
├── package.json
├── src/
│   ├── app/
│   ├── components/
│   ├── pages/
│   ├── services/
│   └── styles/
├── public/
└── tests/
```

## Desarrollo

```bash
npm install
npm run dev
```

## Verificacion

```bash
npm run build
npm test
```

La interfaz sigue las reglas de `design.md`: primera pantalla operativa, componentes consistentes, estados semanticos y layout responsive para el centro de mando KDD.
