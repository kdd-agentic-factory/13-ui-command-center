# Threat Models

This directory contains threat models for each major component of the ecosystem.

## Methodology

Each threat model follows STRIDE:

- Spoofing
- Tampering
- Repudiation
- Information Disclosure
- Denial of Service
- Elevation of Privilege

## Components

- platform-threat-model.md: overall platform
- agent-orchestrator-threat-model.md: agent orchestration
- mcp-gateway-threat-model.md: MCP tool gateway
- rag-cag-threat-model.md: knowledge layer
- skills-autoskills-threat-model.md: skills registry
- kdd-data-pipelines-threat-model.md: data pipelines
- race-command-center-threat-model.md: command center
- race-ai-copilot-threat-model.md: AI copilot
- digital-twin-threat-model.md: digital twin
- infra-docker-threat-model.md: Docker infrastructure
- infra-kubernetes-threat-model.md: Kubernetes infrastructure
- ci-cd-supply-chain-threat-model.md: CI/CD and supply chain
